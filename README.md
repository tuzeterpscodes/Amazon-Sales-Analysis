# Amazon-Sales-Analysis
📊 Amazon Sales Analysis Dashboard Data project evaluating global e-commerce. Cleaned transactional data to find trends, seasonality, and promo impacts. 💡 Insights:  Category market share &amp; price-to-revenue drivers  Regional density via letter-value plots  Geospatial sales intensity heatmaps 🛠️ Tech: Python (Pandas, NumPy), Matplotlib, Seaborn
